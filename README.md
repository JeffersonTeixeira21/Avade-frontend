# My Next.js Application

This project is a web application built using Next.js, designed for educational evaluation and feedback. It allows students to evaluate their teachers and provides administrative functionalities for managing users and viewing performance graphs.

## Features

- User authentication for administrators and students.
- Registration forms for students and teachers.
- Evaluation forms for students to assess teachers.
- Performance graphs based on evaluations.
- Responsive design using Tailwind CSS.

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd my-nextjs-nextapp
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```

## Project Structure

- **app/**: Contains the main application pages and layout.
- **components/**: Contains reusable components such as Navbar, RatingRadio, Toast, and ChartWrapper.
- **lib/**: Contains utility functions and in-memory data storage.
- **styles/**: Contains global CSS styles.

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- Chart.js
